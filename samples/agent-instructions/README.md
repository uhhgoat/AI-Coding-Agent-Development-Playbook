# Agent Instruction Samples

Use these samples as starting points for repository and module instruction
files.

- `root-AGENTS.md` is a repository-wide standing-rules sample.
- `module-AGENTS.md` is a nested module or subsystem sample.

They are templates for the kinds of facts an agent needs, not a required
structure. Rename sections, remove irrelevant rows, and point to existing docs
when a project already has a canonical place for the information.
