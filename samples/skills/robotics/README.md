# Robotics Skills

Reserved for generalized robotics, ROS, SLAM, perception, and hardware-safety
skill examples.

Robotics skills often include high-risk physical-system guidance. Keep examples
generic, conservative, and explicit about simulation, dry-run, operator
approval, emergency stop, and live validation boundaries.

Do not publish robot names, lab layouts, network addresses, calibration values,
customer sites, or hardware procedures that belong to a specific deployment.
